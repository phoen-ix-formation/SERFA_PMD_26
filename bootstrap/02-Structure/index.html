FAQ

Bootstrap est un framework CSS open-source créé par Twitter en 2011. Il fournit un système de grille responsive, des composants UI pré-stylés (boutons, navbar, modales...) et des utilitaires CSS, permettant de créer rapidement des interfaces web professionnelles.

Bootstrap propose des composants prêts à l'emploi (navbar, cards...) avec un design prédéfini. Tailwind est utility-first : chaque propriété CSS est une classe, vous composez tout vous-même. Bootstrap est plus rapide à prendre en main, Tailwind offre plus de flexibilité et de contrôle.

Plusieurs approches : 1) Variables CSS custom (--bs-primary: #votreCouleur dans votre CSS), 2) Compilation SASS en surchargeant les variables Bootstrap, 3) Utiliser data-bs-theme pour le mode clair/sombre. La méthode SASS est la plus robuste pour un projet en production.

Oui, Bootstrap est entièrement gratuit et open-source, sous licence MIT. Vous pouvez l'utiliser dans vos projets personnels et commerciaux sans restriction. Il existe des thèmes et templates premium payants basés sur Bootstrap, mais le framework lui-même est libre.
